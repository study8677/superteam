# 贡献指南

感谢你愿意改进这个教程仓库。

## 可以贡献什么

- 修正不清楚、空泛或容易误解的表达。
- 补充真实团队案例。
- 优化模板，让它更容易直接使用。
- 增加 AI 协作中的风险、反例和检查清单。
- 改进目录结构和阅读路径。

## 内容标准

提交内容时请尽量满足：

- 有明确使用场景。
- 能落到流程、模板、清单或示例。
- 不只宣传工具能力。
- 能区分 AI 负责什么、人负责什么。
- 能说明如何验证结果。

## 不建议的贡献

- 单纯工具排行。
- 没有场景的提示词堆砌。
- 只讲愿景、不讲落地步骤。
- 鼓励绕过审核、隐私或权限边界的做法。

## 提交流程

1. 先开 issue 描述你想改进的问题。
2. 修改对应文档或模板。
3. 提交 PR 时说明：改了什么、为什么改、如何验证可读性。
